JSON = JavaScript Object Notation

🟢 Normal JavaScript Object
const note = {
  title: "Study",
  desc: "React"
};
🔵 JSON version (string)
'{"title":"Study","desc":"React"}'

👉 Notice:

Keys in quotes
Whole thing is a string


🎯 Why JSON exists?

👉 Because:

Browser storage only stores strings
APIs send data as text

So we need a way to convert:

Object ↔ String
🔁 Two Important Functions
🟢 1. JSON.stringify()

👉 Converts object → string

const obj = { title: "A" };

JSON.stringify(obj);

👉 Result:

'{"title":"A"}'
🔵 2. JSON.parse()

👉 Converts string → object

const str = '{"title":"A"}';

JSON.parse(str);

👉 Result:

{ title: "A" }
🔥 Connect to Your App
🧩 Saving to localStorage
localStorage.setItem("notes", JSON.stringify(notes));

👉 Why?

notes = array/object
localStorage needs string
🧩 Loading from localStorage
const saved = localStorage.getItem("notes");
JSON.parse(saved);

👉 Why?

saved = string
You need real array back
🎬 Full Flow
notes (object)
   ↓ stringify
"notes" (string in storage)
   ↓ parse
notes (object again)
⚠️ Important

If you don’t use JSON.parse:

const saved = localStorage.getItem("notes");

👉 You’ll get:

"[{...}]"

❌ Not usable as array

🧠 One Line Understanding

JSON.stringify → object to string
JSON.parse → string to object
